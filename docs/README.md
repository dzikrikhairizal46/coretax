## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+
- npm atau yarn
- Database (SQLite untuk development)

### Installation
```bash
# Clone repository
git clone https://github.com/yourusername/coretax.git
cd coretax

# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Setup database
npm run db:push

# Start development server
npm run dev
```
